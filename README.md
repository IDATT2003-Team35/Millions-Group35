# Millions — Stock Trading Simulation Game

A stock trading simulation game developed for IDATT2003 (Programming 2) at NTNU by Group 35.

---

## What is this?

Players buy and sell stocks on a simulated exchange, manage a portfolio, and try to grow their wealth over multiple in-game weeks. Stock prices fluctuate randomly each week, and realistic commissions and taxes apply to all trades.

---

## Getting Started

### Prerequisites
- Java 25
- Maven 3.x

### Run
```bash
mvn javafx:run
```

### Test
```bash
mvn test
```
